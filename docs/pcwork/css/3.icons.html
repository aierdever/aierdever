图标 Icons

bujichongpc框架pc框架 , css基础

# 放置目录及引用

在样式表中 定义 图标字体名为 soicon

# 目录

static
 ├── css //css样式
 │   ├── soicon //图标集
 │   │   ├── demo.css 
 │   │   ├── demo_index.html //查看所有图标的demo页面
 │   │   ├── iconfont.css //主图标样式文件
 │   │   ├── iconfont.js
 │   │   ├── iconfont.json
 │   │   ├── iconfont.ttf //图标字体文件
 │   │   ├── iconfont.woff
 │   │   ├── iconfont.woff2
 │   ├── base.css //基础样式文件
 │   ├── easy-plus.css //easyui 修正优化样式文件
 │   ├── socss.css // socss框架样式

 ...

# 引用

/* socss.css */
@import './soicon/iconfont.css?v=d59a0';

...

/* class以 icon- 开头的dom对象会被样式命中 */
[class^="icon-"], [class*=" icon-"] {
font-family: "soicon" !important;
/*font-size: 16px;*/
font-style: normal;
speak: never;
font-style: normal;
font-weight: normal;
font-variant: normal;
text-transform: none;
line-height: 1;
-webkit-font-smoothing: antialiased;
-moz-osx-font-smoothing: grayscale;
}

...
